## GUIs für Git

# Beispiele für GUIs für Git sind:
1. Sourcetree
• Kostenlos und viele Funktionen
• Gute Übersicht über Branches & Commits
2. GitKraken
• Sehr übersichtliche grafische Darstellung
• Ideal für Teamarbeit
3. TortoiseGit
• Integration direkt im Windows-Explorer
• Sehr stabil und zuverlässig
4. Fork
• Schnell und moderne Oberfläche
• Geeignet für Anfänger und Fortgeschrittene

### Welches GUI gefällt mir?
Es ist kostenlos, einfach zu bedienen und gut für Anfänger. Die Oberfläche ist übersichtlich und man kann Commits, Push, Pull und Branches leicht machen. Es läuft auf Windows, macOS und Linux und ist direkt mit GitHub verbunden und weil ich es benutzt habe
